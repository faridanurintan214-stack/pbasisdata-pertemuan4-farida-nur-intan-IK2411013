## Deskripsi
Project ini berisi implementasi Stored Procedure dan CASE pada MySQL untuk:
- Status stok barang
- Perhitungan diskon
- Predikat mahasiswa
- Klasifikasi data produk

## Cara Menjalankan
1. Buka MySQL / phpMyAdmin / VS Code
2. Jalankan file SQL
3. Gunakan perintah berikut:

CALL cek_status_stok(3);
CALL hitung_diskon('750.000');
CALL cek_predikat_mahasiswa(85);

## Kesimpulan
IF-THEN-ELSE efektif untuk logika percabangan, CASE praktis untuk klasifikasi data, dan MySQL mampu mengolah data secara logis selain sebagai penyimpanan.
